Create Tasks In Things Using Do Notes

A few weeks ago I wrote about how to chaining Things and web automation service IFTTT. This workaround comes because I have to use PC at work and since my choice app of GTD is Things then I need to sort this problem out. Even I can pick up my iPhone and write down the task directly to Things inbox, it’s not a pleasant things to do.

I have Drafts installed on my iPhone and since it’s a place where text start on iOS, I mainly used it to save article for a blog post ideas. I also use Drafts for do brain dump and send to Things inbox later but what about if I want to quickly capture ideas that suddenly pop-up and I need to get out of it from my mind soon? This is where Do Note by IFTTT take a role.

With my impulsiveness, my mind is unpredictable. Sometimes an idea pop-up in the wrong place and time. To get this idea out of my mind as quickly as possible, I use Do Note. I only need a recipe that makes a task/idea in iOS reminders. Things have reminders integration so whenever I take a note using Do Note and send them to iOS reminders, Things will pick and save it to its inbox. One thing you should pay attention is you have to choose the same iOS reminder list between Do Notes and Things.

The only caveats using this method is sometimes the task/ idea that’s made using Do Note doesn’t appear automatically neither in iOS reminder or Things. I guess it needs to be synced first before the task/ ideas appear.

I know that Drafts could make multiple to-dos to Things inbox and even you can add notes but for an ideas/ tasks that need to get out soon, you can use Do Notes just like I do.
