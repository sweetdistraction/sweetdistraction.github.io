The NEW 1WRITER WITH JAVASCRIPT ACTIONS ∞

Philip Gruneich di OneTapLess:

1Writer 2.0 was the most delightful surprise this week and probably the best deal you could make this holiday if you write in your iOS devices. I’m sure it is only a matter of time until more actions pop up and improve the pool of options in this outstanding text editor.

1Writer, salah satu teks editor di iOS baru saja menerima pembaharuan besar. Penambahan javascript action, pengamanan tambahan dengan Touch ID, kustomisasi font serta tampilan, safari extensions, in-app-browser serta today widget untuk mengakses dokumen favorit secara cepat dan masih banyak lagi.

Ngoc Luu juga menambahkan action directory bagi pengguna yang ingin menambahkan javascript action dengan mudah ke dalam aplikasi 1Writer. Tak hanya itu, bagi pengguna yang ingin belajar membuat javascript action sendiri, Ngoc Luu juga sudah menyiapkan dokumen pendukung yang dapat dipelajari dan digunakan.

Seperti biasa, Philip Gruneich juga telah membuat artikel mengenai update dari 1Writer di OneTapLess dengan contoh javascript action-nya.
