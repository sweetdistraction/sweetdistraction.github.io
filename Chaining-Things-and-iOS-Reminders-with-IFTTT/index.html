Chaining Things and iOS Reminders With IFTTT

GTD or Getting Things Done® is a stress free productivity technique based on David Allen which is awesome and many of the nerds talk about it. It’s simply take 5 steps to GTD: capture, clarify, organize, reflect and engage. Sounds easy, huh?

I knew it last year. Since then I tried to get many references and tools to support GTD systems. I still failed. Sometimes I still fall into cracks, sometimes I let my lists outdated, and the next week I rebuild it.

I’ve tried Omnifocus, plain text based Taskpaper, Things for my task management system to implement GTD and the last one is my choice. Cultured Code the developer behind Things make a great software even the development pace is too slow according to some people.

I love using Things because of its simplicity. The syncing is also fast and reliable. The ease of using Things is there’s no learning curve instead of GTD itself. I don’t said that Omnifocus isn’t a great tools but I get myself overwhelm everytime I open Omnifocus and try to use it.

One features that makes me envy as Things user from Omnifocus is MailDrop features which you can use to send a task to your inbox just by emailing it. Why I envy this features is the fact that I have to use PC at work and there’s such Things for Windows to easily put the task in.

Luckily there’s service called IFTTT which we can use to mimic Omnifocus’ Maildrop features. Things have Reminders integrations which can import all the task in the reminder to its inbox. Now all you have to do is just set up a email trigger in IFTTT that would add a body email as a task with preferred tags in subjects to iOS Reminders. I have Things list in my iOS Reminders that Things ready to import. So, all I need is to set up a recipe that will be triggered by an email with #thingstodo as tag in subjects and a task which I want to add in Things in body email. You can get the IFTTT recipe here.

Now wherever something pops out in my mind while I’m working at work, I just compose email with that tags and send them. At lunch, I will open Things and the list is ready to import.

Sometimes, the recipes don’t start automatically and I have to open IF apps but at least, I become worry less to forget what’s important during works.
